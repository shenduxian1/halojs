    概述

    Installation

    首先将源码下下来
    git clone https://github.com/o2team/haloDoc.git haloDoc

    安装依赖

    cd haloDoc
npm install
git submodule update --init --recursive

    Usage

    新建文章
    hexo new post {{ title }}

    生成静态文件
    hexo g

    开启服务
    hexo s

    打开http://0.0.0.0:4000看效果
